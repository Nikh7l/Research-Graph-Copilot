# Evaluation

## Gold Questions

1. What changed in agent tool-call reliability between 2025-01-01 and 2026-03-22?
2. Which methods for reducing tool-call errors appear most often in the corpus?
3. Which papers discuss structured outputs as a reliability technique?
4. Compare retry-based mitigation versus verification-based mitigation.
5. Generate a briefing on recent approaches to tool-call failure prevention.
6. What evidence exists for planning as a reliability strategy?
7. Which papers mention tool routing or tool selection quality?
8. Which methods have weak evidence and should be treated cautiously?

## Pass Criteria

- Answers cite source papers.
- Date-bounded questions return date-bounded answers.
- Sparse evidence cases return uncertainty rather than overclaiming.
- Briefings include citations and method clusters.
